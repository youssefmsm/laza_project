# Laza - E-commerce Mobile App 

A minimalist e-commerce MVP built with Flutter & Firebase for the Mobile App Development course (Fall 2025).

## Project Overview
Laza is a functional e-commerce app that allows users to browse products, view details, manage a cart, and save favourites. The app uses **Firebase** for backend services and **Platzi Fake Store API** for product data.

## Features
- **User Authentication:** Sign up & Login using Firebase Auth (Email/Password).
- **Product Catalogue:** Fetches real-time data from Platzi API.
- **Cart System:** Add/remove items with persistent storage in Firestore.
- **Favourites:** Save items to a wishlist stored in Firestore.
- **Search:** Local filtering for products.

## Tech Stack
- **Frontend:** Flutter (Dart)
- **Backend:** Firebase (Auth, Firestore)
- **API:** Platzi Fake Store API
